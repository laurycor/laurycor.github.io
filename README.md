## Featured Projects

### NBA Three-Point Analysis
A Tableau dashboard exploring how three-point shooting relates to winning in the NBA from 2000 to 2023.  
The analysis shows that shooting efficiency is a stronger predictor of success than shot volume.

🔗 https://laurycor.github.io

---

### AWS Cloud Computing Project (NYC 311 Data)
A cloud-based data workflow using Amazon S3, AWS Athena, and SageMaker to process and analyze NYC 311 complaint data and build a modeling dataset.

🔗 https://github.com/laurycor/aws-nyc311-laurycor
